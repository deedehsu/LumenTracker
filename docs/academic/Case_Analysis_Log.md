## 判決分析 1：臺灣士林地方法院 115 年度審簡字第 249 號刑事判決
**日期**: 115.04.17
**案由**: 詐欺取財、洗錢

### 1. 案情與金流軌跡摘要
*   **詐欺手法**：通訊軟體 LINE 交友詐騙轉假投資（暱稱「鄭品真」）。
*   **金流軌跡 (兩層洗錢)**：
    *   第一層 (法幣)：被害人匯款（共 5 筆，合計 81,000 元）至被告林振崇的「中華郵政帳戶」。
    *   第二層 (虛擬貨幣)：被告透過網路銀行，將贓款轉匯至其名下 MAX 交易所（現代財富科技有限公司）綁定的「遠東銀行虛擬帳戶」進行入金，購買 USDT 後，再提領轉入詐騙集團指定的不詳電子錢包。

### 2. 法庭上的數位證據攻防
*   **檢方證據**：
    *   告訴人對話紀錄截圖。
    *   中華郵政帳戶開戶資料、存款交易明細、網路銀行登入 IP 紀錄。
    *   **MAX 交易所的申請資料與交易紀錄**（證明遠銀虛擬帳戶為 MAX 入金帳戶，以及購買 USDT 並提領至外部錢包的行為）。
*   **防禦策略**：被告自白犯罪，未對數位證據提出抗辯。檢方原本起訴「三人以上共同詐欺」，但法官認為卷內無數位證據能證明被告認識除「鄭品真」以外的第三人，基於罪疑唯輕原則，改判普通詐欺取財罪。

### 3. 法官的心證與採證標準
*   法官完全採納了 MAX 交易所與銀行提供的交易明細作為洗錢路徑的鐵證。
*   認定被告具備「不確定故意」，因一般人應知悉代收資金購買虛擬幣轉出，常為詐騙集團隱匿金流之手段。
*   **沒收難題**：法官指出，依現有證據，「無法證明被告仍得支配或處分其所轉匯之款項」，因此若宣告沒收實屬過苛，未併宣告沒收（此凸顯了追蹤末端電子錢包歸屬權的實務困境）。

### 4. 與本論文之連結 (LumenTracker 切入點)
*   **實務痛點**：本案金流在進入 MAX 交易所變成 USDT 並提領後，判決書對該筆 USDT 的最終去向僅記載「轉入不詳之人所有之電子錢包」，偵查線索似乎在鏈上中斷。
*   **LumenTracker 價值**：若警方能使用 LumenTracker，即可將 MAX 交易所回函中的提領哈希值 (TxHash) 輸入系統，啟動 `Target Profiler` 自動追蹤該「不詳錢包」，建立幣流關聯圖，進而確認該錢包是否為混幣器、或最終流入哪個海外交易所，突破「斷點」，並產出符合 ARRJ 規範的追蹤報告，或許能改變法官對於「無法支配款項」的心證。

## 判決分析 2：臺灣苗栗地方法院 115 年度苗簡字第 74 號民事判決
**日期**: 115.04.17
**案由**: 侵權行為損害賠償 (民事)

### 1. 案情與金流軌跡摘要
*   **詐欺手法**：二次詐騙（案中案）。原告先遭假投資詐騙，後在臉書尋求「代客追討詐騙款項」，又遭假冒的追討公司（盛世金融、勃發金融）詐騙。
*   **金流軌跡 (三角詐欺)**：
    *   被告鄭、張兩人本身也是詐欺被害人，為了「墊付手續費」或「製造金流」以追回款項，交出自己帳戶（中信銀、國泰世華）。
    *   詐騙集團指示原告將款項匯入被告的帳戶。
    *   被告收到款項後，再依詐騙集團指示，提領現金或以「無卡存款」方式匯入詐欺集團指定的終端帳戶。

### 2. 法庭上的數位證據攻防
*   **原告主張**：被告提供帳戶又轉帳，構成共同侵權行為與不當得利。
*   **被告抗辯 (數位證據)**：提出 LINE 對話紀錄截圖，證明自己是受「盛世金融」欺騙才提供帳戶及幫忙轉匯，缺乏幫助詐欺的故意。

### 3. 法官的心證與採證標準
*   法官採信被告的 LINE 對話紀錄與帳戶明細（顯示帳戶仍有正常生活支出，非專門賣給詐團的人頭帳戶）。
*   認定被告主觀上無過失，且款項已轉出未受利益，駁回原告的損害賠償與不當得利請求。

### 4. 與本論文之連結 (LumenTracker 切入點)
*   **實務痛點**：本案雖然是純法幣的「無卡存款」斷點，但凸顯了「資金過水節點（車手/人頭）」的審認難題。
*   **論文論述延伸**：在虛擬貨幣洗錢中，也常出現不知情的「過水錢包」。LumenTracker 的「雙欄對照表」與「自動化初勘」，正是為了解決這種歸因難題。若應用於幣流，LumenTracker 能藉由計算「過水錢包的進出時間差」與「Gas 來源」，協助法官在民事與刑事庭上，科學地客觀判斷該錢包持有者是否為「有犯意聯絡的洗錢共犯」還是「不知情的受害者」。
## 判決分析 3：臺灣新北地方法院 115 年度金簡字第 79 號刑事判決
**日期**: 115.04.17
**案由**: 詐欺取財、洗錢

### 1. 案情與金流軌跡摘要
*   **詐欺手法**：假投資詐騙 (ZEVICO APP)。
*   **金流軌跡**：
    *   第一層 (法幣)：被害人匯款 75,000 元及 90,000 元至被告張玉宸的「臺北富邦商業銀行」帳戶。
    *   第二層 (虛擬貨幣)：被告依詐欺集團成員「曉瑗」指示，將富邦帳戶內款項轉匯至其名下 MaiCoin 及 MAX 交易所帳戶，兌換成虛擬貨幣後，轉入「曉瑗」指定的「不詳帳戶 (電子錢包)」。

### 2. 法庭上的數位證據攻防
*   **檢方證據**：網路銀行交易明細、告訴人對話紀錄截圖、被告與「曉瑗」的 LINE 對話紀錄。
*   **被告抗辯**：被告坦承提供帳戶並兌換虛擬幣轉出，但辯稱是為了取回自己遭詐騙的款項，才依指示「美化帳戶金流」。
*   **法庭判定**：被告自白犯罪，未對數位證據提出抗辯。

### 3. 法官的心證與採證標準
*   法官認定被告為取回遭詐騙款項，輕率依指示兌換虛擬貨幣並轉匯，製造不法所得金流斷點，具有幫助詐欺、洗錢之不確定故意。
*   **沒收難題**：法官指出，被害人遭詐騙之 16 萬 5,000 元已依指示轉匯，未實際支配占有，宣告沒收過苛，故不予沒收。（再次凸顯追蹤斷點後資金去向的重要性）

### 4. 與本論文之連結 (LumenTracker 切入點)
*   **實務痛點**：本案再次展現「法幣入金交易所 $\rightarrow$ 轉出虛擬幣至未知錢包」的典型洗錢斷點。檢警查到 MaiCoin/MAX 後，往往難以繼續追蹤。
*   **LumenTracker 價值**：LumenTracker 的「自動化初勘」能快速分析被告轉出的終端錢包，判斷其是否為個人錢包或混幣器/交易所熱錢包，有助於突破現有偵查在「境內交易所出金」環節的瓶頸。

## 判決分析 4：臺灣新竹地方法院 114 年度訴字第 1491 號刑事判決
**日期**: 115.04.17
**案由**: 三人以上共同詐欺取財 (面交車手)

### 1. 案情與金流軌跡摘要
*   **詐欺手法**：假投資詐騙 (HIM-PRO 平台)。詐團指示被害人申辦電子錢包 (TLRSxWxjxYyaDAtJvnSepYHt73gabzkziG)。
*   **金流軌跡 (場外交易 OTC)**：
    *   被告朱炫彥擔任「面交車手」，佯裝幣商與被害人面交，收取現金 97 萬 6,800 元。
    *   被告收取現金後，將等額 USDT 轉入被害人的 TLRS 錢包。
    *   (隱藏金流)：被害人以為虛擬幣在自己錢包，實際上早已受詐團控制，後續可能被轉移至投資平台或詐團其他錢包。

### 2. 法庭上的數位證據攻防
*   **檢方證據**：告訴人與詐欺集團的 LINE 對話紀錄、現場監視器錄影翻拍照片。
*   **法庭判定**：被告自白，並繳回犯罪所得 6,000 元。

### 3. 法官的心證與採證標準
*   法官認定被告雖非親自實行詐術，但依指示擔任車手、佯裝交易並轉交贓款，與詐團有犯意聯絡及行為分擔，論以共同正犯。

### 4. 與本論文之連結 (LumenTracker 切入點)
*   **實務痛點**：本案揭示了「假幣商面交」的洗錢手法。被害人雖然拿到了 USDT，但錢包 (TLRS...) 的私鑰可能早已被詐團掌握。
*   **LumenTracker 價值**：LumenTracker 的「涉案幣流關聯畫布」能以被害人錢包 (TLRS...) 為起點，追蹤後續的 USDT 流向。若能證明該錢包內的資金最終流向已知的詐騙集團群集錢包，將更有力地證明該「幣商」與詐欺集團的共犯結構。

## 判決分析 5：臺灣新竹地方法院 114 年度訴字第 1697 號刑事判決
**日期**: 115.04.17
**案由**: 三人以上共同詐欺取財 (面交車手、釣魚扣押)

### 1. 案情與金流軌跡摘要
*   **詐欺手法**：假投資詐騙 (ZentraX 平台)。被害人前次已面交 280 萬元購買 USDT 並投入平台。
*   **金流軌跡 (釣魚面交)**：
    *   詐團要求被害人再投入 30 萬顆 USDT 才能解凍資金。
    *   被害人報警配合釣魚，與假幣商 (被告Ａ０２) 約定面交 900 萬元。
    *   被告抵達現場收取現金回報後，詐團成員「幣財」將 286,624 顆 USDT 轉入被害人錢包。
    *   警方當場逮捕被告，並「扣押」該批 USDT。

### 2. 法庭上的數位證據攻防
*   **爭議焦點 (虛擬貨幣沒收與發還)**：被害人主張扣案的 28 萬顆 USDT 是自己受騙的贓款，聲請發還。
*   **警方溯源分析 (關鍵數位證據)**：警方分析扣案 USDT 來源，發現詐團使用錢包「TMTFF4Vdv3GLTzVdAwLS6kkcyUb3NXGNaH」轉入。回溯發現該錢包還接收了其他多名被害人遭詐騙後層轉的資金，且未發現本案被害人將 USDT 轉入該地址的紀錄。

### 3. 法官的心證與採證標準
*   法官採信警方的鏈上溯源分析，認定扣案的 USDT 來自詐騙集團的其他詐欺所得混同，並非直接來自本案被害人先前被騙的資金。
*   **沒收裁定**：依《組織犯罪防制條例》將該批 USDT 視為「犯罪組織之財產」宣告沒收，駁回被害人發還的聲請（需待判決確定後由執行檢察官分配）。

### 4. 與本論文之連結 (LumenTracker 切入點)
*   **實務痛點與技術展示**：本案極具代表性！警方的「溯源分析報告」直接影響了法院對數百萬財產的沒收與發還裁定。
*   **LumenTracker 價值**：LumenTracker 正是為了產出這類「溯源分析報告」而生。我們的系統能自動畫出資金匯聚到「TMTFF4...」錢包的拓撲圖，並列出多名被害人的資金來源。這份判決強烈佐證了 LumenTracker 的產出在司法實務上的高度需求與證據價值。
