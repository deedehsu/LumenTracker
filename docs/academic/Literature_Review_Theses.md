# 核心學位論文文獻分析 (Thesis Literature Review)

本文件紀錄針對 LumenTracker 專案所蒐集之三篇核心碩博士論文的深度剖析，旨在提取其法學論述與實務痛點，並融入本專案的文獻回顧與架構設計中。

## 論文一：《刑事案件數位證據之跨境調取—美國法之探討與我國法之反思》
* **作者**：方泰鈞
* **年份**：2025 (114年)
* **學位**：國立陽明交通大學 科技法律研究所 碩士
* **核心議題**：探討網際網路資料跨越國界所帶來的取證困難，分析美國 CLOUD Act (雲法案) 及美英雙邊協定，並反思台灣在國際刑事司法互助上的現行法制與困境。
* **LumenTracker 結合點 (DSR 切入點)**：
  * **管轄權困境的技術突圍**：這篇論文點出了傳統法制在跨國取證上的無力。區塊鏈金流(特別是跨國交易所)正是典型的跨境數位證據。LumenTracker 透過 API 自動抓取與本地化保存，等於是用「技術手段」直接取得開源情報，繞過漫長的司法互助請求，大幅提升第一線偵查效率。

## 論文二：《數位證據研究 -以刑事法為中心-》
* **作者**：吳志勇
* **年份**：2025 (114年)
* **學位**：中央警察大學 警察政策研究所 博士
* **核心議題**：從刑事法角度全面檢視數位證據的本質、證據能力與證明力。作者在謝誌中特別強調了具備「與眾不同的專業」以及「拆屋簷式的批判」精神，針對不合理的法律規定提出建言。
* **LumenTracker 結合點 (證據能力與標準化)**：
  * 作為警大博士論文，其對第一線執法與法庭要求的落差必有深刻探討。
  * **ARRJ 與 ISO 27037 的對話**：此論文的論述可作為 LumenTracker 強調的「監管鏈 (Chain of Custody)」與「正當程序 (Justifiability)」的國內最高學術背書。LumenTracker 內建的 Audit Logger 與防呆機制，正是為了回應這種對數位證據嚴謹性的法學要求。

## 論文三：《區塊鏈技術運用於數位證據之研究 —以刑事數位證據之蒐集與鑑識為例》
* **作者**：魯志遠
* **年份**：2020 (109年)
* **學位**：中原大學 財經法律學系 碩士
* **核心議題**：探討數位證據具備「無限複製性、無差異性」的特徵，容易被污染。論文提出運用區塊鏈的「不可竄改、不可否認、永久保存」特性，建立司法證據區塊鏈 (認證、存證、驗證)，提升證據能力。
* **LumenTracker 結合點 (證據固化機制)**：
  * **反向思考的應用**：此論文探討的是用區塊鏈來「保存」證據；而 LumenTracker 則是處理「區塊鏈本身作為證據」的解析。
  * **Hash 固化的學術支持**：論文中提到數位證據易遭竄改的問題。LumenTracker 在 UI 6/7 操作完成後，未來將產出的 PDF 報告附加 SHA-256 Hash 值，正是響應了這篇論文對於「數位證據存證與驗證」的核心呼籲。
