什麼是區塊鏈 API Key？

區塊鏈的資料是公開的，但要讀取這些資料，我們需要透過特定的通道（節點供應商）。

如果您使用我們內建的通道，您的來源可能會與其他調查員混淆，甚至面臨頻寬限制。

因此，我們將引導您申請一組**免費且專屬於您**的 API Key。

API Key 申請教學

我們將引導您申請一個免費的 API Key。以下以 Etherscan 為例：

  1. 前往 etherscan.io 註冊帳號。
  2. 登入後，點擊個人介面的 **API-KEYs**。
  3. 點擊「**Add**」建立一組新的金鑰，並將那串英數字複製下來。
