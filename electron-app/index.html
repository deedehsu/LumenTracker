什麼是區塊鏈 API Key？

區塊鏈的資料是公開的，但要讀取這些資料，我們需要透過特定的通道（節點供應商）。

如果您使用我們內建的通道，您的來源可能會與其他調查員混淆，甚至面臨頻寬限制。

因此，我們將引導您申請一組**免費且專屬於您**的 API Key。

配置您的 API Keys

請申請以下 API Keys 以解鎖 LumenTracker 的完整分析功能。您至少需要配置一個才能繼續。

1. Etherscan API Key (必備，支援以太坊及 EVM 兼容鏈)

請前往 etherscan.io 註冊並新增金鑰。

2. Tronscan API Key (建議，支援波場鏈與 TRC-20 USDT 分析)

請前往 tronscan.org 登入並在「開發者」中產生。

設定主密碼

為了確保您的 API Key 和其他敏感資料的安全，我們建議設定一個**主密碼**。

此密碼將用於使用高等級加密保護您的 API Key，並在每次啟動軟體時要求輸入。

密碼不一致或未填寫！

API Key 儲存失敗，請重試。
