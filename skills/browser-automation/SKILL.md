# browser-automation - 瀏覽器自動化技能

## 描述
此技能旨在提供自動化網頁瀏覽、內容擷取和互動的能力。主要應用於文獻資料的自動搜尋、判決書內容的抓取與結構化，以及需要透過網頁介面進行資訊收集的任務。

## 功能
- **網頁內容擷取**：從指定 URL 抓取網頁的文本或特定元素內容。
- **表單自動填寫與提交**：自動填寫網頁表單並提交。
- **點擊與導航**：模擬用戶點擊網頁元素、執行頁面導航。
- **網頁截圖**：對指定網頁或特定區域進行截圖。
- **搜索結果解析**：針對特定司法或學術網站的搜索結果進行解析與結構化。

## 技術棧
- Puppeteer (或 Playwright) - 用於無頭瀏覽器控制。
- Python - 作為技能的主要實現語言。

## 安全注意事項
- 在執行任何網頁自動化操作前，必須確保目標網站的合法性與操作的合規性。
- 嚴禁進行任何未經授權的數據爬取或攻擊行為。
- 敏感資訊（如登入憑證）的處理必須符合資安規範，不應硬編碼於腳本中。
